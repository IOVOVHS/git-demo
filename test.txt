git init 初始化一个文件夹为工作区
git status 查看当前文件状态
git reflog 查看版本精简信息
git log 查看版本详细信息
git add <fill name> 提交文件到在暂存区 即进行跟踪
git commit -m <log> <file name> 将在暂存区文件提交到本地仓库，形成历史版本
git reset --hard <版本号> 切换到别的版本文件

分支操作
git branch <分支名> 创建分支
git branch -v  查看分支
git check out <分支名> 切换分支
git merge <分支名> 把指定的分支合并到当前分支上 hot-fix
